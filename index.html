Besoin d'aide ?

Si vous avez des difficultés à venir, nous pouvons vous soutenir financièrement. Nous proposons quelques bourses entre 50€ et 250€ selon vos dépenses.

Pour recevoir de l'aide, envoyez nous un email avec:

  • une estimation de vos coûts pour venir (voyage, hébergement, nourriture...),
  • ce qui vous intéresse dans Elm (une ou deux lignes suffisent).

N'ayez pas peur de nous contacter si vous pensez que ce soutien peut vous aider. Les candidatures sont anonymes quelle que soit leur issue.

La keynote

Par Diane Manière

Diane est développeuse et cofondatrice d'une startup lyonnaise qui utilise Elm en production depuis bientôt 3 ans. Elle parlera de plusieurs effets de bord que l'adoption d'Elm a eus sur sa startup. Certains facilement traçables : comment le code CSS a disparu des radars pour sa plus grande joie, comment ses exigences de qualité de code ont évolué; d'autres plus insidieux : comment la programmation fonctionnelle a petit à petit influencé sa façon de coder, ou comment le backend a dû s'adapter à cet engouement croissant pour Elm.
